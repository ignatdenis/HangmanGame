## Hangman Game - C# WinForms
A classic Hangman game implementation developed using C# and Windows Forms. This project was created to practice UI event handling, resource management, and basic game logic.

### Features
Interactive UI: Uses Windows Forms for a graphical user interface.

Dynamic Visuals: The hangman drawing updates progressively with each mistake using custom image assets.

Game Logic: Tracks mistakes, handles character guessing, and monitors the game state (win/loss).

Resource Management: Optimized image loading through the file system.

### Technologies Used
Language: C#

Framework: .NET 8.0 (Windows)

IDE: Visual Studio 2022

### How to Run
Clone the repository.

Open Hangman_Ignat_Denis.sln in Visual Studio.

Press F5 to build and run the application.
